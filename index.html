JADE

Work in progress compiler inspired by JAI

View on GitHub Download .zip Download .tar.gz

I thought Jon Blow's language JAI was interesting, so I started writing a LLVM compiler for it

The language isn't formally defined anywhere yet, as it's still in heavy flux. Jon posts new demos and coding videos semi regularly on YouTube here: https://www.youtube.com/user/jblow888/playlists

But don't get excited...

I don't actually know what I'm doing so this project is mostly a slow moving learning exercise. Right now it's largely a bunch of unit tests hoping to one day generate running code.

Hungarian? In 2015?

Yeah, I know this isn't a popular style, but it's the style used at my work and it's what my brain is used to. If it makes you feel better it's "Apps hungarian" not the "Systems hungarian" used by Microsoft in the 90's. (I know, you still don't like it.)
